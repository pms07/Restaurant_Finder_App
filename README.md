## Restaurants Finder App - Vue.Js

This is Vue.JS app for Restaurants Finder With CSV data with multiple search filters

## Installations

~~~
- Git Clone repo
- Go To root directory of application
- Run Index.html (No Dependency Pull required !!)
~~~
